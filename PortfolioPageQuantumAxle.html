Overview

1st place in the MixJam9 Submission MixJam9 Results Page

Created as participation for the MixJam9, this uses a generator software to develop 3 prompt in which the participants have to create a game based on those prompts.

  1. Robots
  2. Time Travel
  3. Skateboards

Collectively we decided to create an endless runner, originally inspired by Sonic The Hedgehog 2's Chaos Emerald minigames.

This was also my first time creating a game using Unreal Engine with the language set to Japanese. So this provided another experience to learn.

Responsibilities

  • Creating the game design (Game rules, mechanics, players, enemies, level progression).
  • Tile Design
  • Managing the team.
  • Asset placement.
  • UI/UX. (UI animations)
  • All content developed via Blueprints
  • Implementing new assets and having them alter when we warp through time.

Synopsis/Demo

A Robot on a narly adventure through space and time, perhaps he can teach the folks of the past some radical tricks, or show those higher up future folk some retro vibes.

Character Mechanics

  1. Jump/Perform tricks: Player can push the space button multiple times to perfom a couple of tricks increasing their score.
  2. Abduction Counter: If caught in a UFO's tractor beam the player can mash A & D to escape, allowing them to continue their journey.

Enemy Mechanics

Cowboy/Alien

  1. Stationary enemies that fire a single projectile towards the player.

Horse-Riding Cowboy

  1. Will move forward in a straight line, forcing the player to move sideways.
  2. Will fire a projectile towards the player.

UFO

  1. Will attempt to abduct the player. (Player can mash A and D to try to escape.)

New Skills Developed

  • First Games Jam using Japanese in Unreal Engine 5
  • UI Animations.
  • Tile Creation for endless runners
  • Timelines/sockets for grind rails.
  • Implementing distance and score values that increase as the game progresses.
